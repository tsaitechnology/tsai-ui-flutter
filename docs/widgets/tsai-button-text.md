# TsaiButtonText

Inter button-label typography in `large` and `medium` sizes.

[Open live example](https://tsaitechnology.github.io/tsai-ui-flutter/example/#/typography/button-text){ target="_blank" rel="noopener" .md-button }

```dart
const TsaiButtonText(
  'Continue',
  size: TsaiButtonTextSize.large,
)
```

Use `TsaiButton` for interactive actions. `TsaiButtonText` is intended for
custom compositions that need the same typography role.
