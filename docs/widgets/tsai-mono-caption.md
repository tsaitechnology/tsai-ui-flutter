# TsaiMonoCaption

JetBrains Mono caption text with `regular` and `medium` weights.

[Open live example](https://tsaitechnology.github.io/tsai-ui-flutter/example/#/typography/mono-caption){ target="_blank" rel="noopener" .md-button }

```dart
const TsaiMonoCaption(
  '09:41:27 UTC',
  weight: TsaiTextWeight.medium,
)
```

Use this role for compact timestamps, identifiers, and other short technical
metadata.
